# Security Policy

## Supported Versions

Since vyx is currently in early development, only the latest commit on `main` is supported.

| Version | Supported |
|---------|----------|
| `main` (latest) | ✅ |
| older commits | ❌ |

## Reporting a Vulnerability

**Please do not open a public GitHub issue for security vulnerabilities.**

If you discover a security vulnerability in vyx, please report it responsibly:

1. **Email**: Send a detailed report to the maintainer via GitHub's private contact.
2. **Include**: description of the vulnerability, steps to reproduce, potential impact, and any suggested fix.
3. **Response time**: You will receive an acknowledgment within **48 hours** and a resolution plan within **7 days**.

We take security seriously. Thank you for helping keep vyx safe.
